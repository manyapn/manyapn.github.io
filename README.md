# manya pradeep narayan — personal site

Personal site built with Next.js 16, Tailwind CSS, and Framer Motion.

## stack

- **Next.js 16** (App Router, Turbopack)
- **Tailwind CSS v4**
- **Framer Motion**
- Fonts: Dela Gothic One, M PLUS Rounded 1c, IBM Plex Mono

## routes

| path | page |
|---|---|
| `/` | home |
| `/now` | what i'm up to |
| `/experiences` | timeline of work + activities |
| `/projects` | things i've built |
| `/2cents` | opinions |

## dev

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
